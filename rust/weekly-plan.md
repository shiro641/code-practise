# Rust 每周学习计划

本文件由周日 6:00 自动化维护，用来控制“一周能学完”的 Rust 知识点范围。

## 规则

- 每周日生成或更新本周计划。
- 本周计划优先包含上周未完成的 `pending` / `planned`。
- 每周知识点数量控制在 3 到 5 个。
- 知识点按优先级排序：简单、基础、常用优先；用户反复追问的知识点优先。
- 每日 6:00 只从本周计划中抽取一个未掌握知识点。
- 如果存在 `pending`，每日自动化只提醒继续学习该知识点，不推送新的知识点。
