## 当前水平判断

从已有记录看，你已经具备动态规划入门到初级偏上的水平：

- 能把「最后一块石头的重量 II」识别成 0-1 背包，并知道倒序遍历是为了保证每个元素只用一次。
- 能写出一维背包 `dp[j] = Math.max(dp[j], dp[j - weight] + value)` 这类转移。
- 能写出区间 DP 的状态定义、初始化和倒序外层遍历，例如最长回文子序列的 `dp[i][j]`。
- 对「子串」和「子序列」的区别已经被纠正并完成了对应题目。

主要薄弱点：

- 背包容量边界的语义还需要更稳定，例如「不超过 sum / 2」时应优先使用 `Math.floor(sum / 2)`。
- 目前更像是能解出典型题，下一步要训练「看题识别模型」：子集和、完全背包、状态机、区间 DP、二维路径 DP。
- 需要继续要求自己先说出状态定义、转移、遍历顺序和边界条件，再写代码。

建议每日题难度：LeetCode 中等偏易到中等。优先顺序：

1. 0-1 背包变体识别
2. 完全背包入门
3. 股票状态机 DP
4. 区间 DP 巩固
5. 二维路径/网格 DP

## 每日练习记录规则

每日 06:00 的练习会话应读取本目录：

1. 根据 `progress.json` 和已完成题型判断当前水平。
2. 如果上一题还没有完成，不推新题，只提醒继续完成上一题。
3. 如果上一题已完成，给出一道符合当前水平的新 DP 题。
4. 用户提交答案后，评判状态定义、转移、遍历顺序、边界条件和复杂度。
5. 评判通过后，将题目、答案、评价写入新的 `problems/NN-slug/`，并更新 `progress.json`。
