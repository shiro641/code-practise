题目：最后一块石头的重量 II（Last Stone Weight II）

有一堆石头，每块石头的重量都是正整数。

每一回合，从中选出任意两块石头，然后将它们一起粉碎：

如果两块石头重量相同，那么两块石头都会被粉碎；

如果重量分别为 x 和 y，且 x <= y，那么重量为 x 的石头会完全粉碎，而重量为 y 的石头新重量变为 y - x。

最后，最多只会剩下一块石头。

返回此石头最小可能的重量。如果没有石头剩下，就返回 0。

示例 1
输入：stones = [2,7,4,1,8,1]
输出：1

解释：

一种可能的过程：

2 和 4 -> 2
7 和 8 -> 1
2 和 1 -> 1
1 和 1 -> 0
最后剩余 1
示例 2
输入：stones = [31,26,33,21,40]
输出：5
限制
1 <= stones.length <= 30
1 <= stones[i] <= 100

const num = [31, 26, 33, 21, 40];
const getMaxCount = (num: Array<number>) => {
    const sum = num.reduce((acc, cur) => acc + cur, 0);
    const bag = Math.ceil(sum / 2);
    const dp = new Array(bag + 1).fill(0);
    for (let i = 0; i < num.length; i++) {
        for (let j = bag; j >= num[i]; j--) {
           //  dp[j] = num[i] + dp[j - num[i]] > bag ? dp[j - num[i]] : num[i] + dp[j - num[i]]
           dp[j] = Math.max(dp[j-num[i]]+num[i], dp[j])
        }
    }
    // console.log(dp, bag)
    return dp[bag]
}
const part = getMaxCount(num)

你这个解法是对的，已经把它成功转成了 0-1 背包。

不过有一个小优化点：

const bag = Math.ceil(sum / 2)

更常见写法是：

const bag = Math.floor(sum / 2)

因为我们要找的是：

不超过 sum / 2 的最大子集和

然后答案是：

sum - 2 * dp[bag]

你这里倒序遍历也写对了，因为每块石头只能用一次，是 0-1 背包。
