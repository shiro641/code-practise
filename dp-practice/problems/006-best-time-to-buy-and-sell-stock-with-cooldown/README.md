# 006. 最佳买卖股票时机含冷冻期

## 状态

已完成。

## 题目

给定一个整数数组 `prices`，其中 `prices[i]` 表示第 `i` 天股票的价格。

你可以多次买卖一支股票，但同时最多只能持有一股。卖出股票后，你无法在第二天立刻买入股票，也就是存在 1 天冷冻期。

请返回你能获得的最大利润。

## 示例

```text
输入：prices = [1,2,3,0,2]
输出：3
解释：对应交易过程可以是 [买入, 卖出, 冷冻, 买入, 卖出]
```

```text
输入：prices = [1]
输出：0
```

## 限制

```text
1 <= prices.length <= 5000
0 <= prices[i] <= 1000
```

## 核心识别

这是股票状态机 DP。

- `dp[i][0]` 表示第 `i` 天结束后未持有股票的最大利润。
- `dp[i][1]` 表示第 `i` 天结束后持有股票的最大利润。
- 冷冻期的关键不是“`i - 2` 那天买入”，而是“如果第 `i` 天买入，那么前一个可转移来源只能是第 `i - 2` 天结束时未持有”。

## 你的解答

见 `solution.ts`。

关键问答记录：

```text
1. 为什么 dp[i][1] 不能从 dp[i - 1][0] - prices[i] 转移，而要考虑 dp[i - 2][0] - prices[i]？
答：因为有冷冻期，第 i 天若发生买入，前一天不能是刚卖出后的未持有状态，所以买入来源不能直接接在第 i - 1 天未持有之后。

2. 为什么 dp[1][1] 不能直接写成 -prices[1]？
答：因为还要考虑第 0 天买入后到第 1 天继续持有，所以要在第 0 天买入和第 1 天买入之间取最大值。

3. 为什么最后返回最后一天未持有状态即可？
答：因为持有状态代表手里还有股票，没有把利润兑现；在题目不限制卖出的前提下，最终最优答案一定落在未持有状态。
```

## 评价

通过。

代码逻辑正确，关键点都过关：

- 边界条件正确：`prices.length <= 1` 返回 `0`，`prices.length <= 2` 时直接处理。
- 初始化正确：`dp[0][0] = 0`，`dp[0][1] = -prices[0]`，并且 `dp[1][1]` 已修正为同时考虑“第 0 天买入后继续持有”和“第 1 天直接买入”。
- 状态转移正确：

```ts
dp[i][0] = Math.max(dp[i - 1][0], dp[i - 1][1] + prices[i]);
dp[i][1] = Math.max(dp[i - 1][1], dp[i - 2][0] - prices[i]);
```

- 冷冻期理解基本到位。需要再压实的一句话是：`dp[i - 2][0] - prices[i]` 表示“第 `i` 天买入时，上一个合法来源是第 `i - 2` 天结束后未持有”，不是“第 `i - 2` 天买入”。

```text
时间复杂度：O(n)
空间复杂度：O(n)
```
