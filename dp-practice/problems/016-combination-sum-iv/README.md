## 题目

给定一个由互不相同的正整数组成的数组 `nums`，和一个目标整数 `target`。请你从 `nums` 中选取若干个数，使它们的和等于 `target`，并返回不同排列的总数。

注意这里统计的是排列数，不是组合数。

## 用户解答

```python
input = [1,2,3]
target = 4


def getCountNum(input, target):
    dp = [0] * (target + 1)
    dp[0] = 1

    for j in range(target+1):
        for i in range(len(input)):
            if j<input[i]:
                continue
            dp[j] = dp[j-input[i]] + dp[j]

    return dp[target]


if __name__ == "__main__":
    print(getCountNum(input, target))
```

补充说明：

- 一开始把题目算成了组合计数。
- 后来意识到如果 `i` 放在外层，会把数字顺序固定住，转移出来的是组合数。
- 改成外层枚举目标和、内层枚举可选数字后，`dp[j]` 才能表示“和为 `j` 的排列数量”。

## 评价

通过。

这次关键进步不是只把代码改对，而是你把“为什么之前错、现在为什么对”说清楚了。

- 状态定义正确：`dp[j]` 表示凑成和为 `j` 的不同排列数量。
- 边界条件正确：`dp[0] = 1`，表示和为 0 时有 1 种空排列，作为后续拼接的起点。
- 状态转移正确：如果最后一个放入的数是 `num`，那么前面部分必须凑成 `j - num`，所以有 `dp[j] += dp[j - num]`。
- 遍历顺序正确：外层枚举目标和 `j`，内层枚举可选数字 `num`。这样是在按“最后一步放什么”累加，天然统计到不同顺序的排列。
- 复杂度正确：时间复杂度 `O(target * len(nums))`，空间复杂度 `O(target)`。

你这次的核心辨析已经到位：

- 外层枚举物品、内层枚举容量，通常更接近组合计数。
- 外层枚举容量、内层枚举物品，才能让每个位置都重新尝试所有数字，从而统计排列。
