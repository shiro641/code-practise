## 题目

给你一个只包含正整数的非空数组 `nums`。

请你判断是否可以将这个数组分割成两个子集，使得两个子集的元素和相等。

## 示例

```text
输入：nums = [1,5,11,5]
输出：true

解释：
可以分割成 [1,5,5] 和 [11]
```

```text
输入：nums = [1,2,3,5]
输出：false
```

## 限制

```text
1 <= nums.length <= 200
1 <= nums[i] <= 100
```

## 分类

0-1 背包 / 子集和判定。

典型思路：

```text
sum 为奇数，直接 false。
target = sum / 2。
dp[j] 表示是否能用前面若干数字凑出和 j。
对每个 num，j 从 target 倒序到 num：
dp[j] = dp[j] || dp[j - num]
```
