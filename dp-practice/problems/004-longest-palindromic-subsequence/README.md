# 004. 最长回文子序列

## 状态

已完成。

## 题目

给你一个字符串 `s`。

请你找出其中最长的回文子序列，并返回该序列的长度。

## 示例

```text
输入：
s = "bbbab"

输出：
4
```

```text
输入：
s = "cbbd"

输出：
2
```

```text
输入：
s = "abcde"

输出：
1
```

## 限制

```text
1 <= s.length <= 1000
s 仅由小写英文字母组成
```

## 易错点

题目问的是最长回文子序列，不是最长回文子串。

```text
子串：必须连续
子序列：可以不连续，但相对顺序不能变
```

例如 `"bbbab"` 的答案是 `4`，因为可以选择不连续的 `"bbbb"`。

## 评价

这版是对的。

状态定义：

```ts
dp[i][j]: s[i...j] 区间内的最长回文子序列长度
```

转移：

```ts
if (s[i] === s[j]) {
  dp[i][j] = dp[i + 1][j - 1] + 2
} else {
  dp[i][j] = Math.max(dp[i + 1][j], dp[i][j - 1])
}
```

你的额外判断：

```ts
i + 1 > j - 1 ? 2 : dp[i + 1][j - 1] + 2
```

可以安全处理长度为 2 的区间，比如 `"bb"`。在 JS/TS 里这样写比直接访问越界位置更稳。
