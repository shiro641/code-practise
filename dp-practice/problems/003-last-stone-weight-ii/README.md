## 状态

已完成。

## 题目

有一堆石头，每块石头的重量都是正整数。

每一回合，从中选出任意两块石头，然后将它们一起粉碎：

- 如果两块石头重量相同，那么两块石头都会被粉碎。
- 如果重量分别为 `x` 和 `y`，且 `x <= y`，那么重量为 `x` 的石头会完全粉碎，而重量为 `y` 的石头新重量变为 `y - x`。

最后，最多只会剩下一块石头。返回此石头最小可能的重量。如果没有石头剩下，就返回 `0`。

## 示例

```text
输入：stones = [2,7,4,1,8,1]
输出：1
```

```text
输入：stones = [31,26,33,21,40]
输出：5
```

## 限制

```text
1 <= stones.length <= 30
1 <= stones[i] <= 100
```

## 核心识别

最后剩下的重量可以看作两堆石头总重量之差。目标是让一堆尽量接近 `sum / 2`，转化为 0-1 背包。

```ts
const bag = Math.floor(sum / 2)
```
