# 009. 一和零

## 状态

已完成。

## 题目

给你一个二进制字符串数组 `strs` 和两个整数 `m`、`n`。

请你找出并返回 `strs` 的最大子集长度，该子集中最多有 `m` 个 `0` 和 `n` 个 `1`。

## 示例

```text
输入：strs = ["10","0001","111001","1","0"], m = 5, n = 3
输出：4
解释：最多可以选出 4 个字符串，比如 {"10","0001","1","0"}
```

```text
输入：strs = ["10","0","1"], m = 1, n = 1
输出：2
解释：可行子集包括 {"10"} 和 {"0","1"}，其中最大子集长度为 2。
```

## 限制

```text
1 <= strs.length <= 600
1 <= strs[i].length <= 100
strs[i] 只由 '0' 和 '1' 组成
1 <= m, n <= 100
```

## 核心识别

这是 `0-1 背包` 的双维容量变体。

- 每个字符串只能选或不选一次，所以本质上仍然是 `0-1 背包`。
- 这次背包容量不再是一维，而是两维：`0` 的数量限制和 `1` 的数量限制。
- 目标不是求是否可行，也不是求方案数，而是求“最多能选多少个字符串”，所以是最值型 DP。

## 你的解答

你先补充了建模：

```text
dp[j][k] 表示使用 j 个 0、k 个 1 时，子串集合的最大长度。
```

```text
每次将最新的 strs[i] 纳入考虑后，需要考虑两个策略：放入 strs[i] 和不放入。
放入的情况下，最大数量就是 dp[j - count0][k - count1] + 1；
不放入的话，就维持上一轮的 dp[j][k]；二者取最大。
```

```text
dp[0][0] 表示使用 0 个 0 和 0 个 1，此时无法放入任何子串。
```

## 评价

通过。

状态定义正确：`dp[j][k]` 表示在当前处理到前若干个字符串时，使用不超过 `j` 个 `0`、不超过 `k` 个 `1` 时，最多能选出的字符串数量。

转移正确：

```ts
dp[j][k] = Math.max(dp[j][k], dp[j - count0][k - count1] + 1);
```

含义是：不选当前字符串，则保持 `dp[j][k]`；选当前字符串，则从剩余容量 `j - count0`、`k - count1` 的最优解转移过来，再加上当前这个字符串。

遍历顺序正确：`j` 和 `k` 都倒序遍历，保证当前字符串在一轮转移中只会被使用一次；如果正序遍历，就会在同一轮里重复利用当前字符串，变成完全背包语义。

边界条件正确：二维数组初始化为 `0`，表示在任何容量下，如果一个字符串都不选，最多都能得到长度 `0` 的子集；`dp[0][0] = 0` 只是这个初始化中的一个特例。

复杂度：

```text
时间复杂度：O(len(strs) * m * n)
空间复杂度：O(m * n)
```
