# 012. 买卖股票的最佳时机 II

## 题目

给定一个整数数组 `prices`，其中 `prices[i]` 表示某支股票第 `i` 天的价格。

在每一天，你可以决定是否买入和/或卖出股票。你在任意时刻最多只能持有一股股票，但你可以在同一天先买入再卖出。

返回你能获得的最大利润。

## 示例

```text
示例 1：
输入：prices = [7,1,5,3,6,4]
输出：7
解释：第 2 天买入，第 3 天卖出，利润 4；第 4 天买入，第 5 天卖出，利润 3；总利润 7。
```

```text
示例 2：
输入：prices = [1,2,3,4,5]
输出：4
```

```text
示例 3：
输入：prices = [7,6,4,3,1]
输出：0
```

## 限制

```text
1 <= prices.length <= 3 * 10^4
0 <= prices[i] <= 10^4
```

## 核心识别

这是 `股票状态机 DP` 的基础题。

- 每天结束时，状态只需要区分“手里不持股”和“手里持股”。
- 因为允许无限次交易，但同一时刻最多持有一股，所以不需要记录交易次数维度。
- 这题虽然也能用贪心解释，但当前训练重点是先把它识别成状态机 DP。

## 你的解答

你提交的是 Python 实现：

```python
prices_ = [1,2,3,4,5]

def getMaxCount(prices):
    dp = [[0] * 2 for _ in range(len(prices))]
    # print(dp)

    dp[0][0] = 0
    dp[0][1] = -prices[0]

    for i in range(1, len(prices)):
        # print(i)
        dp[i][0] = max(dp[i - 1][1] + prices[i], dp[i - 1][0])
        dp[i][1] = max(dp[i - 1][0] - prices[i], dp[i - 1][1])
    
    print(dp)
    return dp[len(prices)-1][0]


if __name__ == "__main__":
    print(getMaxCount(prices_))
```

## 评价

通过。

从你的代码可以反推出状态定义是正确的：

- `dp[i][0]` 表示第 `i` 天结束后，手里 `不持有股票` 时的最大利润。
- `dp[i][1]` 表示第 `i` 天结束后，手里 `持有股票` 时的最大利润。

转移也正确：

```text
dp[i][0] = max(dp[i - 1][0], dp[i - 1][1] + prices[i])
dp[i][1] = max(dp[i - 1][1], dp[i - 1][0] - prices[i])
```

含义分别是：

- 今天结束时不持股，要么昨天就不持股，今天什么都不做；要么昨天持股，今天卖出。
- 今天结束时持股，要么昨天就持股，今天什么都不做；要么昨天不持股，今天买入。

遍历顺序正确：

- 从第 1 天开始顺序遍历到最后一天。
- 因为 `dp[i]` 只依赖 `dp[i - 1]`，所以按天从前往后推就可以。

边界条件也正确：

- `dp[0][0] = 0`：第 0 天结束时不持股，利润为 0。
- `dp[0][1] = -prices[0]`：第 0 天结束时持股，说明第 0 天买入了一次。

复杂度正确：

```text
时间复杂度：O(n)
空间复杂度：O(n)
```

如果继续优化，还可以把二维 `dp` 压成两个变量，把空间降到 `O(1)`。但按当前题目要求，你这份写法已经完全成立。
