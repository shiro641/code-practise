# 021. 最小路径和

- 日期：2026-06-28

- 类型：二维路径 / 网格 DP
- 状态：已完成

## 题目

给定一个包含非负整数的 `m x n` 网格 `grid`，请找出一条从左上角到右下角的路径，使路径上的数字总和最小。

说明：

- 每次只能向下或者向右移动一步。

## 示例

```text
示例 1：
输入：grid = [[1,3,1],[1,5,1],[4,2,1]]
输出：7
解释：路径 1 → 3 → 1 → 1 → 1 的总和最小。
```

```text
示例 2：
输入：grid = [[1,2,3],[4,5,6]]
输出：12
```

## 限制

```text
m == grid.length
n == grid[i].length
1 <= m, n <= 200
0 <= grid[i][j] <= 100
```

## 用户解答

你提交的是建模答案：

```text
dp[i][j] 表示到达位置 i，j 时的最小总和
状态转移：dp[i][j] = min(dp[i-1][j], dp[i][j-1]) + grid[i][j]
遍历顺序：i 从 1 到 m-1，j 从 1 到 n-1（从上到下、从左到右）
初始化：dp[0][0] = grid[0][0]
dp[0][x] = dp[0][x-1] + grid[0][x]
dp[y][0] = dp[y-1][0] + grid[y][0]
最终答案：dp[m-1][n-1]
复杂度：时间 O(mn)，空间 O(mn)
```

你的这份建模已经完整且正确：

- 状态定义准确：`dp[i][j]` 表示从左上角走到 `(i, j)` 时的最小路径和。
- 转移正确：当前位置只能从上方或左方过来，因此取两者较小值再加当前格子。
- 遍历顺序正确：按行从上到下、按列从左到右，能保证依赖的上方和左方状态先算出来。
- 边界条件正确：第一行只能从左边过来，第一列只能从上边过来，`dp[0][0] = grid[0][0]`。
- 复杂度正确：时间复杂度 `O(mn)`，空间复杂度 `O(mn)`。
